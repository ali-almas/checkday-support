FAQ

  • Calendar not showing? Enable access in iOS Settings → Privacy & Security → Calendars.
  • Meeting transcription not working? Allow Microphone and Speech Recognition permissions.
  • AI summaries run only when you explicitly request them.
  • Subscriptions are managed by Apple via the App Store.

Privacy

Checkday works primarily on-device. If you request an AI summary, meeting transcript text is sent to OpenAI’s API for processing. We do not store user data on our servers.
